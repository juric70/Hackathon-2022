NodeJS REST API tehnička dokumentacija za HR Schemu

Općenito

Dokument opisuje NodeJS REST API za HR Schemu i izložene resurse. Namijenjen je za vježbe iz kolegija "Razvoj Naprednih Web Aplikacija" na FSRE. Za korištenje API-ja potrebno je obaviti HTTP zahtjev i obraditi odgovor. Kod poziva se koriste standardne HTTP metode: GET, POST, PUT i DELETE.

Napomena: API je sklon promjenama.
Verzija tehničke dokumentacije možda bude zaostala za verzijom API-ja.

employees

Naziv metode i resursa Opis
GET    employees Dohvat svih zaposlenika.
GET    employee/{employee_id} Dohvat zaposlenika sa traženim {employee_id}.
GET    search/employees?search={query} Dohvat zaposlenika sa first_name ili last_name koji sadrže string {query}.
POST     employee Kreiranje novog zaposlenika sa proslijeđenim objektom u req.body.
PUT    employee Ažuriranje zaposlenika sa proslijeđenim objektom u req.body. Potrebno je navesti i employee_id u objektu
PUT    employee/{employee_id} Ažuriranje zaposlenika sa traženim {employee_id} sa proslijeđenim objektom u req.body.
DELETE   employee/{employee_id} Brisanje zaposlenika sa traženim {employee_id}.
